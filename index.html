China Times

2019-2020

China Times is a round-based WebGL game that aims to reflect the modern development of China. The game is hosted on the USC server, and is only open to USC students.

I maintained this game in 2017 when the game was implemented with Photon Networking. In 2019, many of dependencies of the original Unity project were outdates, so I remade this game completely (UI, game mechanics, and data synchronization). A big design change is that I used a server-client structure instead of a peer-to-peer connection, which improves reliability of the game and reduces difficulty of future development.

Domain

2018-2019

Domain is an Advanced Game Project at USC. This game is a multiplayer location-based mobile game. I was in charge of all server functionalities (e.g. game loop and game stats persistence), stats monitoring, and service endpoints.
